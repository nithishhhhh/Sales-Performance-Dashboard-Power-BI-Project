# Sales Performance Dashboard — Power BI Project

##  Project Overview

This project is an end-to-end Sales Analytics Dashboard built using **Power BI** to analyze business performance across products, regions, customer segments, and time.

The goal of this project is to demonstrate real-world data analysis skills including data modeling, DAX calculations, KPI reporting, and interactive dashboard design.

---

##  Business Problem

Organizations need clear insights into:

* Sales trends
* Profitability
* Regional performance
* Customer segments
* Discount impact

This dashboard provides decision-ready insights for business stakeholders.

---

##  Dataset

Global Superstore dataset containing:

* Orders
* Customers
* Products
* Geography
* Sales transactions

## Data Modeling

Built a **Star Schema Model**:

Fact Table:

* Sales transactions

Dimension Tables:

* Customers
* Products
* Geography
* Date table (custom DAX calendar)

---

##  DAX Measures Created

* Total Sales
* Total Profit
* Profit Margin %
* Year-over-Year Growth
* Average Discount
* Sales YTD
